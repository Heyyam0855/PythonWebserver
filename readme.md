### 🐍 Python Web Server

Bu layihə Python istifadə edərək **interaktiv və modern** bir web server yaratmağı öyrədir.

## 📋 Tələblər
- Python 3.x
- Heç bir əlavə kitabxana tələb olunmur (built-in modullar)

## 🚀 İstifadə

1. Server-i işə salmaq üçün:
```bash
python server.py
```

2. Brauzerinizdə açın:
```
http://localhost:8000
```
